# Contributing to Liseli

Liseli is Zambia's open language project. Like the Human Genome Project mapped every human gene, Liseli is mapping every word, phrase, and sentence in Zambia's 7 official languages — so AI can finally speak them.

This is a national effort. Every contribution matters.

## How to Contribute

### As a Language Speaker

The most valuable contribution is your knowledge of Zambian languages. You can:

- **Translate** — Convert English content into your language
- **Validate** — Review existing translations for accuracy
- **Discuss** — Help resolve disputed translations, document dialect variations

No technical skills needed. If you speak a Zambian language, you can contribute.

### As a Developer

- Fork the repo, make changes, submit a pull request
- Focus areas: UI/UX improvements, data pipeline, offline support, API development
- Run `npm install && npm run dev` to get started locally

### As a Linguist or Researcher

- Help define quality standards for translations
- Identify and document dialect variations
- Use the data for research — and share findings back
- Help structure morphological and grammatical data

### As a Data Contributor

- Know of Zambian language text sources? Open an issue
- Government documents, educational materials, community publications
- Any openly licensed text in Bemba, Nyanja, Tonga, Lozi, Kaonde, Lunda, or Luvale

## Data Principles

1. **Open** — All data is CC BY-SA 4.0. Always.
2. **Community-owned** — No individual or company owns the translations
3. **Quality over quantity** — One verified translation is worth more than a hundred unverified ones
4. **Dialect-aware** — We document variation, we don't enforce a single "correct" form
5. **Attribution** — Contributors are credited for their work

## Code of Conduct

- Respect all languages equally — no language is more important than another
- Respect dialect differences — there is no single "correct" form of any language
- Be constructive in discussions — disagreement about translations is expected and valuable
- No spam, no automated low-quality submissions
